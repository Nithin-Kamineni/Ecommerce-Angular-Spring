# Ecommerce-Angular-Spring
 Ecommerce Clone is a feature-rich web app using Spring Boot, Angular, MySQL, and Java. It enables customers to browse products, add items to their cart, and securely checkout with Stripe. Okta integration ensures safe user authentication. The app is containerized with Docker for easy deployment, while Kafka facilitates efficient event streaming.
